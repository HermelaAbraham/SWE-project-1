# ShareIT Login Page

This is the login screen built using React + Vite for the ShareIT platform.

## Features
- Custom layout based on design mockup
- Responsive styling
- Login form with placeholder inputs

## Getting Started
1. Clone this repo
2. Run `npm install`
3. Run `npm run dev` to launch locally
